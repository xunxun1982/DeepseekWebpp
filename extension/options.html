工具目录

这里展示 DeepSeek 可主动调用的工具。新增工具后，记得同步 Native Host、提示词、权限策略和测试。

web_search 设置

默认使用 DuckDuckGo。Bing 和 DuckDuckGo 无额外参数；MCP 使用 mcpServers 配置。

无需 API

Bing 和 DuckDuckGo 只需要能访问对应网页，无需填写额外参数。

本地 MCP：GrokSearch-rs

这里填写 mcpServers 风格 JSON。`command` 可以是 PATH 中的 `grok-search-rs`，也可以是你自己的绝对路径。

白名单

白名单按工具和作用范围匹配。文件变更类工具仍会每次确认。

运行环境

运行状态和工具结果续写设置保存在浏览器本地数据中。

外观

页面浮窗默认位于右侧中部，可拖动；位置会保存在浏览器本地存储中。
